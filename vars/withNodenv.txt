Install Nodenv and setup multiple versions of Node for the pipeline.

Parameters:
 * __version__ (default: '6.14.4'): set the NodeJS vrsion desired inside the pipeline.
 * __method__ (default: 'keep'): option to keep or delete the version installed.

Syntax:
```
  script {
    withNodenv('9.11.0', 'clean') {
      sh "node --version"
    }
  }
```

Other methods:
 * withNodenv.purgeAll('nodenv') - Method used to cleanup all NodeJS versions installed.
